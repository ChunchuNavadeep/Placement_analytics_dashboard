Placement Analytics Dashboard


📌 Project Overview

The Placement Analytics Dashboard is a data visualization project designed to analyze student placement and internship records. It transforms raw data into meaningful insights using data cleaning, Power BI dashboards, and a simple web UI prototype.

This project helps in understanding hiring trends, internship impact, and department-wise placement performance.

🎯 Objectives
Analyze student placement patterns across departments
Study internship-to-placement conversion trends
Identify top recruiting companies and hiring behavior
Visualize data for better decision-making
✨ Features
 Interactive Power BI dashboards for insights
 Cleaned and structured dataset from raw CSV files
 Company-wise hiring analysis
 Internship vs placement comparison
 Simple HTML-based UI prototype
 Visual screenshots for better understanding
🛠️ Tech Stack
Power BI – Data visualization and dashboard creation
HTML – UI prototype representation
CSV Files – Dataset format
Data Cleaning Tools – Excel / Python (if used)
